# PixelGuy CSS - Guybrush Threepwood Spyglass Animation

A pure CSS pixel art animation featuring Guybrush Threepwood from the Monkey Island series. This project demonstrates the power of CSS by creating intricate pixel art animations using only the `box-shadow` property—no images, no JavaScript, just CSS.

## Demo

Open `index.html` in your web browser to see the spyglass animation in action.

## How It Works

- **Pixel Art with Box-Shadows**: Each "pixel" is a `box-shadow` positioned absolutely to form the character and background.
- **Animation**: Keyframe animations change the `box-shadow` values over time to create the spyglass looking motion.
- **Responsive**: Uses viewport units (`vh`, `vw`) for scaling.

## Files

- `index.html`: The HTML structure with a single `<div class="guybrush">`.
- `style.css`: Contains all the CSS for the pixel art and animation.
- Screenshots: Static images capturing frames of the animation.

Inspired by the classic point-and-click adventure game *The Secret of Monkey Island* and the art of CSS-only graphics.
